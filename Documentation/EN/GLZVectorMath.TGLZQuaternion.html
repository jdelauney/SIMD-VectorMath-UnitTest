type TGLZQuaternion = record

Description

TGLZQuaternion : The quaternions are a number system that extends the complex numbers and applied to mechanics in three-dimensional space.
A feature of quaternions is that multiplication of two quaternions is noncommutative.

function ConvertToMatrix : TGLZMatrix;

ConvertToMatrix : Construct a rotation matrix from (possibly non-unit) quaternion.
Assumes matrix is used to multiply column vector on the left :
vnew = mat * vold.
Works correctly for right-handed coordinate system and right-handed rotations.

function MultiplyAsSecond(const qFirst : TGLZQuaternion): TGLZQuaternion;

MultiplyAsSecond : Returns quaternion product quaternion_Right * quaternion_Left.
To combine rotations, use the product Muliply(qSecond, qFirst), which gives the effect of rotating by qFirst then qSecond.
Note : order is important !
