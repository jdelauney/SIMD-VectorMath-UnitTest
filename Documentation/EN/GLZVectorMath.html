Description :
GLZVectorMath is an optimized vector classes math library for FreePascal and Lazarus
using SIMD (SSE, SSE3, SS4, AVX, AVX2) acceleration
It can be used in 2D/3D graphics, computing apps.

Notes :

Notice, here, we need to set rounding FPC mode to is the same as our SSE code.
In FPC, the "function Round" using "banker's rounding" algorithm.
In fat, is not doing a RoundUp or RoundDown if the is exactly x.50
Examples :
Round(2.5) //return = 2
Round(3.5) //return = 4

For more informations see : https://www.freepascal.org/docs-html/rtl/system/round.html

Record TGLZHmgPlane A plane equation. Defined by its equation A.x+B.y+C.z+D, a plane can be mapped to the homogeneous space coordinates, and this is what we are doing here.
The plane is normalized so in effect contains unit normal in ABC (XYZ) and distance from origin to plane.

WARNING : Create(Point, Normal) will allow non unit vectors but basically DON'T DO IT.
A non unit vector WILL calculate the wrong distance to the plane.

It is a fast way to create a plane when we have a point and a normal without yet another sqrt call.
